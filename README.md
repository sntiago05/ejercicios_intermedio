# Ejercicios Python Intermedio

Menú interactivo para ejecutar los ejercicios de Python desde la terminal.

## Requisitos

- Python 3 instalado
- Una carpeta llamada `soluciones/` con los archivos de ejercicios

## Estructura esperada

```
proyecto/
├── main.py
└── soluciones/
    ├── 1_sabor_mas_pedido.py
    ├── 2_cliente_frecuente.py
    └── ...
```

Los archivos deben seguir el formato `NUMERO_nombre_del_ejercicio.py`.

## Cómo ejecutar

```bash
python3 main.py
```

Se mostrará un menú con todos los ejercicios disponibles:

```
=== MIS EJERCICIOS ===
1 - sabor mas pedido
2 - cliente frecuente

Escribe el número del ejercicio: 
```

Escribe el número del ejercicio que quieras ejecutar y presiona Enter.  
Escribe `0` para salir.
